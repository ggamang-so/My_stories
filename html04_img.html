웹페이지에 이미지 넣기

        img태그 : 이미지 표시 
        확장자: gif, jpg, png, svg ...
        src속성 : 이미지 파일 경로 입력()
                 ./ : 현재위치 ../:상위폴더위치
        alt속성 : 이미지 로딩 안될때 대체텍스트
        title : 풍선도움말
        height : 높이 (그냥 숫자는 픽셀(px), 백분율은 %붙여서)
        width  : 너비 (그냥 숫자는 픽셀(px), 백분율은 %붙여서)

        span태그 : 아무 기능이 없이 다른 속성을 추가 할때 사용한다.
