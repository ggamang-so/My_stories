폼 양식 만들기

        input : 입력 폼 만들기 태그
        name : 이름 / 서버로 데이터가 넘어갈 때 사용하는 변수
        type : 값을 입력할 타입을 설정하는 속성
        size : 입력하는 폼의 크기
        placeholder : 입력 전에 채워넣을 내용
        maxlength : 최대입력 갯수
        disabled : 비활성화
        value : 값설정
        checked : 체크박스에서 미리 체크해 놓기

        get 방식 전송 : url에 form으로 입력된 정보가 노출되며 전송되는 방식
        post 방식 전송 : 입력 내용이 드러나지 않고 전송하는 방식
        hidden속성 : 데이터는 필요하지만 표기하고 싶지 않을때 사용
성별 : 남성 여성
취미 : 축구 야구 탁구 농구 배구
여러줄 택스트 입력 가능
